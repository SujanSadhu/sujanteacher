Q11
Find the distance between the points (–5, 7) and (3, –1).
SHOW YOUR STEPS · USE DISTANCE FORMULA
SOLUTION
d = √[(3−(−5))² + (−1−7)²] = √[8² + (−8)²] = √[64 + 64] = √128 = 8√2 ≈ 11.31 units
Q12

SOLUTION

Q13
Find the coordinates of the point dividing the line joining A(3, –2) and B(9, 10) internally in the ratio 2 : 3.
SOLUTION
Using section formula: x = (2×9 + 3×3)/(2+3) = (18+9)/5 = 27/5 = 5.4
y = (2×10 + 3×(−2))/(2+3) = (20−6)/5 = 14/5 = 2.8
Point: (27/5, 14/5)
Q14
Find the value of k such that the points (1, 3), (4, k) and (7, 9) are collinear.
SOLUTION
Area of triangle = 0 for collinear points.
½|1(k−9) + 4(9−3) + 7(3−k)| = 0
|k−9 + 24 + 21−7k| = 0 → |−6k + 36| = 0 → k = 6
Q15
Find the area of the triangle formed by the points (1, 2), (5, 6) and (9, 2).
SOLUTION
Area = ½|x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂)|
= ½|1(6−2) + 5(2−2) + 9(2−6)|
= ½|4 + 0 − 36| = ½ × 32 = 16 sq. units
Q16
Find the ratio in which the point (3, 4) divides the line joining (1, 2) and (7, 10).
SOLUTION
Let ratio = k : 1. Then x = (7k+1)/(k+1) = 3
7k + 1 = 3k + 3 → 4k = 2 → k = 1/2
Ratio = 1/2 : 1 = 1 : 2
Q17
Find the coordinates of the point which divides the line joining A(–4, 2) and B(8, 14) externally in the ratio 3 : 2.
SOLUTION
External section formula: x = (m·x₂ − n·x₁)/(m−n), y = (m·y₂ − n·y₁)/(m−n)
x = (3×8 − 2×(−4))/(3−2) = (24+8)/1 = 32
y = (3×14 − 2×2)/(3−2) = (42−4)/1 = 38
Point: (32, 38)
Q18
If A(2, 4), B(6, 8) and C(x, y) are vertices of a triangle with area 12 square units, find a relation between x and y.
SOLUTION
Area = ½|x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂)| = 12
|2(8−y) + 6(y−4) + x(4−8)| = 24
|16 − 2y + 6y − 24 − 4x| = 24
|4y − 4x − 8| = 24
4y − 4x − 8 = ±24
∴ y − x = 8 or y − x = −4
Q19
Four points represent corners of a rectangular garden: A(–2,1), B(6,1), C(6,7), D(–2,7).
(i) Find the length and breadth.  (ii) Coordinates of diagonal intersection.  (iii) Length of diagonal AC.  (iv) Verify diagonals are equal.
SOLUTION
(i) Length AB = |6−(−2)| = 8 units; Breadth BC = |7−1| = 6 units

(ii) Diagonals intersect at midpoint of AC or BD:
M = ((−2+6)/2, (1+7)/2) = (2, 4)

(iii) AC = √[(6−(−2))² + (7−1)²] = √[64+36] = √100 = 10 units

(iv) BD = √[(−2−6)² + (7−1)²] = √[64+36] = √100 = 10 units ✓ Diagonals are equal.
Q20
Three observation towers at A(–4,2), B(6,2), C(2,10). Control station P(x,y) is equidistant from A and B.
(i) Find the equation satisfied by P.  (ii) If P lies on line AC, find P.  (iii) Find distance of P from C.
SOLUTION
(i) PA = PB ⟹ PA² = PB²
(x+4)²+(y−2)² = (x−6)²+(y−2)²
x²+8x+16 = x²−12x+36 → 20x = 20 → x = 1
Equation: x = 1 (the perpendicular bisector of AB)

(ii) Line AC: slope = (10−2)/(2−(−4)) = 8/6 = 4/3
Equation: y−2 = (4/3)(x+4) → y = (4/3)x + 22/3
Substitute x=1: y = 4/3 + 22/3 = 26/3
P = (1, 26/3)

(iii) PC = √[(2−1)² + (10−26/3)²] = √[1 + (4/3)²] = √[1 + 16/9] = √(25/9) = 5/3 units
